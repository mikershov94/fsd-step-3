МЕТОДЫ

Функции, которые находятся в объекте в качестве свойств
    называются методами.

Методы позволяют объектам выполнять действия.

    let user = {
        name: 'John',
        age: 22,

        sayHi: function() {
            console.log('Hello');
        }
    };

    user.sayHi();       //вызов метода

Методы могут ссылаться на свой объект через this.

    let user = {
        name: 'John',
        age: 22,

        sayHi: function() {
            console.log(this.name);  //выведет строку из свойства текущего объекта.
        }
    };

Значение this не фиксировано. Оно вычисляется во время выполнения кода.
    Зависит от контекста.

ПРАВИЛО.
    при вызове obj.f() значение this внутри f равно obj. 

СТРЕЛОЧНЫЕ ФУНКЦИИ.
    У стрелочных функций нет своего this.
    Значение this стрелочной функции будет обраться из внешней функции.

МЕТОДЫ У ПРИМИТИВОВ
    Примитивы это значения.
    В JavaScript примитивы не могут быть объектами.

    Чтобы у примитивов можно было запускать методы у примитива создается объект-обертка.
    После использования объект-обертка удаляется

    Пример методов:
        let str = 'Привет';
        str.toUpperCase()

    Объект-обертка создается специальными конструкторами (работают без new)
        Number();
        String();
        Boolean();

    Null и Undefined не имеют методов
