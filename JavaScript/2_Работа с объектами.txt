РАБОТА С ОЬЪЕКТАМИ

Особенность!
    Можно обратиться к любому свойству.
    Даже если его не существует.
    В этом случае вернется значение undefined.

    В следствие этого, можно проверить свойства на существование.

        user.age === undefined

    Лучше использовать специальный оператор

Проверка наличия свойства

    'name' in user;  //вернет true

Перебор свойств

    Для перебора свойств в объекте используется специальный цикл for ... in ...

        for (key in obj) {
            // тело цикла, выполняется для каждого свойства объекта
        }

    Пример.
        let user = {
            name: "John",
            age: 30,
            isAdmin: true
        };

        for (let key in user) {
            // ключи
            alert( key );  // name, age, isAdmin
            // значения ключей
            alert( user[key] ); // John, 30, true
        
        }

Копирование по ссылке.

    Примитивные типы данных присваиваются и копируются по значению

    Объекты копируются по ссылке.
    Это значит что объект находится где-то в памяти.
    При присваивании или копировании используется ССЫЛКА на эту область памяти.
