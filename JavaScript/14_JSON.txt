JSON

JSON (JavaScript Object Notation) – это общий формат для представления значений и объектов. 

JSON легко использовать для обмена данными, когда клиент использует JavaScript, 
    а сервер написан на Ruby/PHP/Java или любом другом языке

Преобразование в JSON

    Используем метод JSON.stringify(obj)
        возвращает JSON-форматированную строку. Ее можно записать в файл.

    JSON.stringify(obj) вызывает у объекта метод toJSON() если он есть

Проеобразование из JSON

    Используем метод JSON.parse(str)
        возвращает объект JavaScript


Для удобства в объекте можно реализовать метод toJSON() с пользовательскими настройками
Он работает по принципу toString()
